Sobre mí

Soy Desarrollador Front-End Junior y Técnico Informático, con experiencia en el desarrollo de interfaces web funcionales, claras y orientadas al usuario, utilizando HTML, CSS, JavaScript y React para crear sitios y aplicaciones web dinámicas.

He trabajado de manera independiente desarrollando sitios web, landing pages y aplicaciones web, integrando formularios, flujos de datos y automatizaciones. Además, utilizo Python para la lógica de backend, creación de APIs con Flask y desarrollo de herramientas internas para gestión de datos y procesos.

Me especializo en transformar necesidades reales en soluciones digitales, combinando diseño, lógica y usabilidad. Tengo experiencia integrando plataformas externas como WhatsApp, formularios y sistemas de contacto, y desarrollando interfaces pensadas para usuarios finales.

También soy Técnico Informático, con más de cuatro años de experiencia en soporte, reparación y mantenimiento de equipos, tanto de forma presencial como remota, lo que me permite comprender los sistemas desde el hardware hasta el software.

Actualmente soy estudiante de Ingeniería Informática en la UNDAV y continúo formándome de manera constante para fortalecer mis habilidades en desarrollo web y tecnologías modernas.

Busco integrarme a un equipo donde pueda crecer como desarrollador front-end, aportar soluciones reales y seguir evolucionando profesionalmente.
